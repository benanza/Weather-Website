Latitude vs. Cloudiness

This is an interesting plot insomuch as it demonstrates the notably high occurrence of particular percentages of cloudiness (0%, 20%, 40%, 75%, 90%, and 100%). However, these values do not appear to have any correlation with distance from the equator. This is verified by the linear regression that was run on the plot that had an insignificant r-squared value.
